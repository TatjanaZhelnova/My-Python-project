# My-Python-project
Basic Python: ETL-process
Процессы:  
1. Обращение к нашему API для получения данных  
2. Обработка и подготовка данных к загрузке в базу данных в PostgreSQL  
3. Загрузка обработанных данных в локальную базу PostgreSQL  
4. Формирование необходимых запросов для агрегации данных за определенный период времени  
5. Агрегирование данных для загрузки в Google Sheeets  
6. Загрузка данных в таблицу Google Sheets
7. Настройка оповещений о результатах процесса по электронной почте Mail.ru (добавлен вариант с отправкой лога)

Для процесса настроен лог работы скрипта с отлавливанием всех ошибок и выводом промежуточных стадий. Лог сохраняется в тектовый файл.  
Файл именуется в соответствии с текущей датой. Если в папке с логами уже есть другие логи - их необходимо удалять, остаются только логи за последние 3 дня.
